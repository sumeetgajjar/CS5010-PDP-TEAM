The stock prices of following tickerNames is available with the program.
"AAPL", "GOOG", "GE", "BAC", "ORCL", "VZ", "MS", "T"

The program only has stock prices of last 100 days from today. Hence while purchasing the stocks
this constraints need to be taken care by the user.

In order to run the jar, cd to the res folder and please execute the following command:
java -jar virtualTrading.jar

P.S: Please maximize the terminal before running the jar
